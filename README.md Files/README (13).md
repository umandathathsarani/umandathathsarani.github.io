# 🌌 StarLog Archive: Deep Space Tracker

A sleek, interactive web application designed to monitor, categorize, and archive reading transmissions. Built for enthusiasts who want a clean, terminal-inspired interface to track their journey through literature.

## 🚀 Live Demo
[View the live archive here](https://binuris-voyage.netlify.app/)

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom animations & flexbox), Vanilla JavaScript
* **Data Storage:** Web Storage API (`localStorage`) for persistent, local-first data tracking
* **Deployment:** Netlify Continuous Deployment
* **Version Control:** Git & GitHub

## 📋 Features
* **Mission Logging:** Initialize new entries with Title, Author, Genre, and specific Mission Notes.
* **Archive Management:** View, update status, and manage your reading list.
* **CRUD Functionality:** * **Create:** Log new transmissions.
    * **Read:** View active and completed missions.
    * **Update:** Toggle mission status (Reading ↔ Completed) and edit notes.
    * **Delete:** Permanently purge transmissions from the database.
* **Telemetry Logs:** Real-time analytical breakdown of your reading progress.
* **Interactive UI:** Features a custom glowing stardust cursor effect and responsive, terminal-styled design.
