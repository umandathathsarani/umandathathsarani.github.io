# Cyberpunk Hacker Escape

🕹️ **Live Demo:** [Play the Game Here](https://cyberpunk-hacker-escape.netlify.app/)

An interactive, text-based terminal web game built to demonstrate vanilla web development skills, state management, and dynamic DOM manipulation.

## Features
* **Branching Narrative:** Multiple endings and complex storyline paths based on user choices.
* **State Management:** An inventory system that tracks items and unlocks specific story options.
* **Custom Minigame:** A custom-built, timing-based hacking mechanic to bypass system firewalls.
* **Immersive UI:** Includes CSS animations, synthesized Web Audio typing sounds, and a dynamic neon cursor.
* **Auto-Save:** Utilizes browser `localStorage` to preserve game state and inventory across sessions.

## Tech Stack
* HTML5
* CSS3 (Modular Architecture, CSS Variables, Keyframe Animations)
* Vanilla JavaScript (ES6+, Web Audio API, Asynchronous Intervals)
