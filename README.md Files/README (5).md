# 🏰 Discrete Mathematics Grimoire (IT1160)

Welcome to the **Discrete Mathematics Grimoire**, a comprehensive digital collection of lecture notes, tutorials, and algorithm analysis exercises for the IT1160 course at the Sri Lanka Institute of Information Technology (SLIIT). 

This project is designed as a "living spellbook" for undergraduate students, providing a structured, easy-to-navigate, and visually immersive way to master complex discrete mathematical concepts.

## 📖 Project Overview
This website serves as a personal academic repository for the Year 01, Semester 02 curriculum, covering topics ranging from Set Theory to Finite Automata.

### 📜 Course Syllabus
The grimoire is organized into the following chapters:
* **Lec 1:** Set Theory
* **Lec 2:** Functions
* **Lec 3:** Sequences
* **Lec 4:** Counting I
* **Lec 5:** Counting II
* **Lec 6:** Induction
* **Lec 7:** Graphs
* **Lec 8:** Trees & Heaps
* **Lec 9:** Recursion
* **Lec 10:** Analysis of Algorithms
* **Lec 11:** Finite Automata

## 🛠️ Tech Stack & Styling
* **Web Structure:** Semantic HTML5
* **Styling:** Custom CSS featuring:
    * Hogwarts-themed "Dark Academia" aesthetic.
    * Glassmorphism effects for layout containers.
    * Custom scrollbar styling.
    * Fully responsive layout with active-tab highlighting.
* **Icons:** Powered by FontAwesome.

## 🚀 How to Run Locally
1. Clone this repository: `git clone https://github.com/umandathathsarani/discrete-math-notes.git`
2. Open the folder in VS Code.
3. Use the **"Live Server"** extension: right-click `index.html` and select **"Open with Live Server"**.
