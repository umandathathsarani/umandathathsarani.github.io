# Stress Relief Hub 🧘‍♀️✨

**Live Demo:** [https://umandathathsarani.github.io/stress-relief-hub/](https://umandathathsarani.github.io/stress-relief-hub/)

The Stress Relief Hub is a comprehensive, interactive single-page web application designed to help users unwind, focus, and de-stress. Built entirely from scratch without heavy frameworks or game engines, this project features 12 custom-engineered, physics-driven modules that combine mesmerizing visuals with procedurally generated ambient audio.

## 🚀 Features & Modules

The application includes 12 unique interactive experiences:

1. **🫧 Bubble Wrap:** Satisfying pop mechanics with audio feedback.
2. **🌱 Zen Garden:** A calming, generative drawing tool mimicking raked sand.
3. **⚔️ Zen Slicer:** Fluid, physics-based slicing mechanics.
4. **⏳ Kinetic Sand:** A real-time cellular automata engine simulating falling sand dunes.
5. **🁣 Domino Topple:** Pathfinding and chain-reaction physics for endless domino falling.
6. **🎨 Spin Art:** Centrifugal force simulation using off-screen canvas rendering.
7. **⌨️ Lo-Fi Keys:** 3D CSS mechanical switches paired with synthesized Solfeggio frequencies.
8. **💧 Water Ripples:** Expanding, overlapping 2D liquid wave simulations.
9. **🌌 Breathing Orb:** A guided 4-7-8 meditation tool synchronized with deep ambient swells.
10. **🌀 Harmonograph:** Intricate geometric art generated by simulated pendulum mathematics.
11. **✨ Firefly Meadow:** Wandering algorithms and flocking (Boids) mechanics.
12. **🎐 Wind Chimes:** 2D pendulum physics with multi-frequency collision audio.

## 🛠️ Technology Stack

This project was built focusing on core web technologies and optimized performance:

* **HTML5 Canvas API:** Used for high-performance 2D rendering, particle systems, and complex path drawing.
* **Web Audio API:** Used for procedurally generated sound effects (synthesizers, oscillators, gain nodes, and biquad filters).
* **Vanilla JavaScript (ES6+):** Object-oriented architecture, custom state management, and a lightweight SPA routing system.
* **CSS3:** Advanced animations, 3D transforms, and fully responsive grid layouts.
