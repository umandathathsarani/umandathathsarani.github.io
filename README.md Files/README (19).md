# Spotify Song Popularity Predictor

## Overview
This project is an end-to-end Artificial Intelligence and Machine Learning solution designed to predict whether a track will become popular based on its audio features (such as Danceability, Energy, Tempo, and Loudness). 

## Dataset Details
* **Source:** Spotify Tracks Dataset (Kaggle)
* **Features:** Audio characteristics including Danceability, Energy, Key, Loudness, Mode, Speechiness, Acousticness, Instrumentalness, Liveness, Valence, and Tempo.
* **Target Variable:** Popularity (0-100 score).

## How to Run the Code
1. Clone this repository to your local machine.
2. Download the 'Spotify Tracks Dataset' from Kaggle and place the `.csv` file in the `data/raw/` directory.
3. Install the required Python packages (Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn).
4. Navigate to the `notebooks/` directory and execute the notebooks in numerical order, or run the `integrated_pipeline.ipynb` for the complete workflow.
