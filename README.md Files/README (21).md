> **Miraculous Timeflow** goes beyond basic time tracking. It integrates mathematical animations, state-machine logic for productivity, and an interactive particle physics engine, all wrapped in a dual-identity UI inspired by Ladybug and Cat Noir.

---

### ✨ Core Features

* **🌍 World Clock & Analog Sync:** Dynamic digital clocks for multiple global timezones alongside a custom-built, mathematically synced analog clock face.
* **⏳ Precision Timer:** Track days, hours, minutes, and seconds with an intuitive digital input interface.
* **🔔 Live Alarm System:** Add, toggle, and delete scheduled alarms with real-time browser alerts.
* **⏱️ Professional Stopwatch:** Millisecond precision tracking coupled with a dynamic, scrollable lap recording system.
* **🧠 Pomodoro State Machine:** Automated cycling through 25-minute focus sessions, short breaks, and long breaks, complete with visual progress tracking dots.
* **🎭 Dual Identity Theme:** Seamlessly toggle between a professional dark UI and a dynamic "Secret Identity" theme.
* **🎨 Custom Canvas Physics:** An interactive, twinkling glitter rain system that falls dynamically and uses 2D collision detection to bounce off the application's UI cards.

---

### 🛠️ Architecture & Technologies

* **Structure:** Semantic `HTML5` with `<canvas>` integration for particle systems.
* **Styling:** `CSS3` utilizing custom variables for theme toggling, flexbox layouts, radial gradients, glowing text-shadows, and responsive media queries.
* **Logic:** Vanilla `JavaScript (ES6+)` managing advanced DOM manipulation, `Intl.DateTimeFormat` API for global timeframes, interval state management, and a custom `requestAnimationFrame` physics engine.
* **Deployment:** Continuous Integration/Continuous Deployment (CI/CD) hosted via **Netlify**.
