# Personal Portfolio Website ✨

**Live Demo:** [https://umandathathsarani.github.io/](https://umandathathsarani.github.io/)

A modern, responsive, and highly dynamic personal portfolio website designed to showcase my journey as an **AI Engineer and ML Researcher**. Built entirely from scratch using fundamental web technologies, this portfolio features a sleek dark mode aesthetic, smooth CSS animations, and an interactive glassmorphism user interface.

## 🚀 Features
* **Modern UI/UX:** Built with a premium dark theme, sleek gradients, and abstract floating background shapes.
* **Responsive Design:** Fully fluid layout utilizing Flexbox, CSS Grid, and a custom mobile side-drawer hamburger menu.
* **Dynamic Animations:** Utilizes the JavaScript Intersection Observer API to trigger smooth fade-in animations as the user scrolls.
* **Project Showcase:** Dedicated sections highlighting featured Machine Learning and Full-Stack developments.

## 🛠️ Tech Stack
* **HTML5:** Semantic structuring and accessibility.
* **CSS3:** Custom properties (variables), Backdrop filters, Media queries, and Keyframe animations.
* **Vanilla JavaScript (ES6+):** Modular DOM manipulation, event delegation, and scroll observation.

## 💻 How to Run Locally
Since this project uses pure HTML, CSS, and JavaScript, no complex build tools or package managers are required.

1. Clone the repository:
   ```bash
   git clone https://github.com/umandathathsarani/umandathathsarani.github.io.git
   ```
2. Navigate into the directory:
   ```bash
   cd umandathathsarani.github.io
   ```
3. Open `index.html` directly in your favorite web browser.
   * *Tip: For the best development experience, open the folder in VS Code and use the **Live Server** extension.*
